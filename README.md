# 极简文本工具站
一款纯前端、本地处理的轻量级文本工具集合，支持 JSON 格式化、文本去重、Base64 加解密、URL 转码等常用功能，**无数据上传、无广告、免费商用**。

## ✨ 核心特性
- 🚀 **纯前端本地处理**：所有文本操作均在浏览器内完成，无数据上传至服务器，保障信息安全；
- 📱 **全端适配**：完美兼容 PC、平板、手机等设备，响应式布局体验一致；
- 🎨 **简洁易用**：界面极简，功能直达，无需注册/登录，开箱即用；
- 📄 **合规商用**：基于 MIT 协议开源，个人/企业可免费商用，无授权限制；
- ⚡ **轻量高效**：无冗余依赖，加载速度快，操作即时响应。

## 🛠 功能列表
| 功能          | 说明                          |
|---------------|-------------------------------|
| JSON 格式化   | 一键格式化/校验 JSON 文本     |
| 文本去重      | 按行去重，保留首次出现顺序    |
| Base64 加解密 | 支持任意文本的 Base64 编解码  |
| URL 转码      | 支持 URL 编码/解码，兼容中文  |

### 本地部署
1. 克隆仓库
```bash
git clone https://github.com/260227/text-tool-site.git
cd text-tool-site
```

2. 安装依赖
```bash
npm install
```

3. 启动开发服务
```bash
npm run dev
```

4. 构建生产版本
```bash
npm run build
```

5. 部署构建产物
将 `dist` 目录部署至任意静态服务器（如 Nginx、GitHub Pages、Netlify 等）。

## 📋 技术栈
- 框架：Vue 3 (Composition API)
- 路由：Vue Router 4
- UI 组件：Element Plus
- 工具库：@vueuse/head（SEO 优化）
- 构建工具：Vite
